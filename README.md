# Speech2RIR

This is the official implementation of reverberant speech to room impulse response estimator. We trained our network using Room Impulse Responses from [**SoundSpaces-NVAS dataset**](https://github.com/facebookresearch/novel-view-acoustic-synthesis/tree/main)
